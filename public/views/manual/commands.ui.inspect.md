## Inspect Object

### Description

Gives a description of an room, and an object.

### Syntax:

	inspect
	inspect{};
	me.inspect{object};

### Parameters

<dl>
  <dt>inspect{}:</dt>
  <dd>Gives description of room.</dd>
</dl>

<dl>
  <dt>object:</dt>
  <dd>Gives description of an object.</dd>
</dl>
