## inventory

### Description

Displays contents in right hand or left hand to the screen. 

### Syntax:

	System.out.println(handVariable);

### Parameters

<dl>
  <dt>inventoryCode</dt>
  <dd>An integer indicating the inventory code for the application.</dd>
