## Door

### Description

When encountering a door you type the correct java code to open the door.

### Syntax

    door.open;
	door.close;


### Parameters

<dl>
  <dt>door:</dt>
  <dd>Is a game object, to access the next room.</dd>
</dl>

<dl>
  <dt>open:</dt>
  <dd>Is used to open any door.</dd>
</dl>

<dl>
  <dt>close:</dt>
  <dd>Is used to close any door.</dd>
</dl>
