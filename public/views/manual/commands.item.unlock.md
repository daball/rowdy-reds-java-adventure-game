## unlock

### Description

The capability to unlock a locked door, or footlocker.

### Syntax

	object.unlock(handvariable);
	footLocker.unlock(handVariable);

### Parameters

<dl>
  <dt>unlockCode</dt>
  <dd>An integer indicating the assigning to unlock code for the application.</dd>
</dl>
