## me.leftHand

### Description

Assigns object to player's left hand.

### Syntax

    leftHand = gameObject;
    me.leftHand = gameObject;

### Parameters

<dl>
  <dt>leftHandCode</dt>
  <dd>An integer indicating the assigning to left hand code for the application.</dd>
</dl>
