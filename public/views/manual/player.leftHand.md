## ASSIGN TO LEFT HAND

### Description

Assigns object to player's left hand.

### Syntax

    leftHand = gameItems;
    me.leftHand = gameItems;

### Variables

<dl>
  <dt>leftHand:</dt>
  <dd>Consist of players left hand.</dd>
</dl>

<dl>
  <dt>me:</dt>
  <dd>Consist of the player.</dd>
</dl>

<dl>
  <dt>gameobject:</dt>
  <dd>Consist of game items.</dd>
</dl>
