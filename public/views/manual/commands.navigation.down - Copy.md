## DOWN

### Description

Navigate down to access lower level rooms, or go down a set of stairs, or fall on the floor.

### Syntax

    D
    DOWN
    moveDown();
    me.moveDown();

### Parameters

<dl>
  <dt>downCode</dt>
  <dd>An integer indicating the down code for the application.</dd>
</dl>
