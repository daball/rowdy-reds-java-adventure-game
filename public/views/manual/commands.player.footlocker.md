## footLocker

### Description

Unlocks the foot locker, assuming you have the correct key in your hand.

### Syntax:

	footLocker.unlock(handVariable);

### Parameters

<dl>
  <dt>footLockerCode</dt>
  <dd>An integer indicating the foot locker code for the application.</dd>
</dl>
