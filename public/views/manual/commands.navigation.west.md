## WEST

### Description

Navigate west to access other rooms, or a wall.

### Syntax

    W
    WEST
    moveWest();
    me.moveWest();


### Parameters

<dl>
  <dt>westCode</dt>
  <dd>An integer indicating the west code for the application.</dd>
</dl>
