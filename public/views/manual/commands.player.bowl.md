## bowl

### Description

Add lamb chop to bowl, assuming you have a lamb chop in the hand you are referring too.

### Syntax:

	bowl = handVariable;

### Parameters

<dl>
  <dt>bowlCode</dt>
  <dd>An integer indicating the bowl code for the application.</dd>
</dl>
