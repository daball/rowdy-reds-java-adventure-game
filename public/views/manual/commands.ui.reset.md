## RESET

### Description

Reset the game to the front Entrance of Castle.

### Syntax

    RESET
    RESTART
    
    

### Parameters

<dl>
  <dt>resetCode</dt>
  <dd>An integer indicating the reset, and restart code for the application.</dd>
</dl>
