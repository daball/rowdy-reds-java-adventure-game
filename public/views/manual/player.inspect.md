## Inspect Object

### Description

Gives a description of an object.

### Syntax:

	me.inspect{gameobject};

### Parameters

<dl>
  <dt>me:</dt>
  <dd>Consist of the player.</dd>
</dl>

<dl>
  <dt>inspect{}:</dt>
  <dd>The current room, and/or game items.</dd>
</dl>

<dl>
  <dt>gameobject:</dt>
  <dd>Consist of game items.</dd>
</dl>
