## me.rightHand

### Description

Assigns object to player's right hand.

### Syntax

    rightHand = gameObject;
    me.rightHand = gameObject;

<dl>
  <dt>rightHandCode</dt>
  <dd>An integer indicating the assigning to right hand code for the application.</dd>
</dl>
