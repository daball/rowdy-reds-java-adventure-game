## Dog

### Description

When encountering the dog, feed the dog the lamb chop so you can access the next room.

### Syntax

    bowl = handVariable;


### Parameters

<dl>
  <dt>dogCode</dt>
  <dd>An integer indicating the dog code for the application.</dd>
</dl>
