## unlock

### Description

Unlock Containers.

### Syntax

	gameItems.unlock{handvariable};
	footLocker.unlock{handVariable};

### Parameters

<dl>
  <dt>gameItems:</dt>
  <dd>Consist of game items.</dd>
</dl>

<dl>
  <dt>unlock{}:</dt>
  <dd>Unlocks any container.</dd>
</dl>

<dl>
  <dt>handVariable:</dt>
  <dd>Choose a hand.</dd>
</dl>

<dl>
  <dt>footLocker:</dt>
  <dd>Consist of a game item.</dd>
</dl>
