## lamp.wind

### Description

When player enters a dark room and holding a lamp. Use this command to light the lamp and illuminate a room.

### Syntax

	lamp.wind();

### Parameters

<dl>
  <dt>lamp:</dt>
  <dd>Consist of a game object.</dd>
</dl>

<dl>
  <dt>wind():</dt>
  <dd>Charge lamp to produce light.</dd>
</dl>
